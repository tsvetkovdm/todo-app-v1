# todo-app-v1

Простое консольное todo-приложение на Go для создания задач и их управления.

## Доступные команды

```text
help	            показать список команд
add {title} {text}	добавить новую задачу
list	            показать все задачи
done {title}	    отметить задачу как выполненную
del {title}	        удалить задачу
events	            показать историю событий
exit	            выйти из программы
```

## Структура проекта
```bash
todo-app-v1/
├── scanner/
│   ├── errors.go   # пользовательские ошибки для ввода и команд
│   ├── event.go    # описание событий и история действий
│   ├── prints.go   # функции для вывода информации в консоль
│   └── scanner.go  # обработка CLI-ввода и разбор команд
├── todo/
│   ├── errors.go   # ошибки, связанные с задачами
│   ├── list.go     # хранение списка задач и операции add/list/done/del
│   └── task.go     # модель задачи
├── go.mod
├── go.sum
├── main.go         # запуск приложения
└── README.md
