# todo-app-v1

Простое консольное todo-приложение на Go для управления задачами

### Доступные команды

```bash
help	            показать список команд
add {title} {text}	добавить новую задачу
list	            показать все задачи
done {title}	    отметить задачу как выполненную
del {title}	        удалить задачу
events	            показать историю событий
exit	            выйти из программы
```
